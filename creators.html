TBOF Creator Info

Specifications

  • Block textures must be 64x64 pixels.
  • Use PNG format with transparency when needed.
  • Textures must tile seamlessly (no borders or artifacts).
  • File naming convention: blockname_variant.png (e.g., stone_smooth.png).
